# 安装python 环境
## 安装 pyenv-virtualenv
使用 Homebrew 安装
```bash
brew install pyenv-virtualenv
```

## 创建隔离的工作环境
```
pyenv virtualenv (python-version) (workspace-name)
```
## 激活环境：
```
pyenv activate (workspace-name)
```
## 退出环境：
```
pyenv deactivate
```
